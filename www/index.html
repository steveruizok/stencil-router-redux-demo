Stencil App Starter

This project is a very simple demo of a stencil-redux working alongside stencil-router. It was creating using: 

npm init stencil 
npm install redux 
npm install @stencil/redux 
npm install redux-thunk 
npm install stencil-router
The store is created in app-root and passed to routes using: 
componentProps={{ store: this.store }}
It works as you'd expect! In this example, we have a state with a single property, count. It's currently at 0, but we can change it using the control below. When we route to our Profile Page, the count should be preserved.
